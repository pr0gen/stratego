# Game Engine

### Dependencies

```
npm install -g nodemon
npm install @types/node --save-dev
```

### Configuration

Copy the **.env.example** and rename to **.env**
```
cp .env.example .env
```
Complete the **.env** file 

### Install

```
npm install
```

### Start Server

```
npm run start
```

### Dev mode

```
npm run dev
```
